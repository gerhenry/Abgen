Why Abgen?

Consistent, reproducible abstracts reduce integration friction and accelerate top-level floorplanning. Abgen fits existing EDA flows with minimal disruption.

Highlights

  • Scriptable flow (CLI + config files)
  • Deterministic pin/obstruction export
  • Layout health checks & reports
  • Works alongside commercial tools

Get started

# clone
git clone https://github.com/gerhenry/Abgen
cd Abgen

# dry run with example config
./scripts/run_abgen.sh --config configs/example.yml --out out/demo

Config-first flow

Drive abstracts from version-controlled YAML. Promote repeatability across teams and tapeouts.

Outputs

Abstract views, pin maps, obstructions, metrics (area, density), and machine-readable reports for CI.

Roadmap

GUI helper, OA/LEF/DEF export options, rule packs, and CI recipes.
